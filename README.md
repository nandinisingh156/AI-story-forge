# AI StoryForge

AI StoryForge is an AI-powered storytelling platform that creates personalized stories using Ollama and Flask.

## Features

* AI Story Generation
* Multiple Genres
* Multiple Story Types
* Knowledge-Based Storytelling
* PDF Download Support
* Responsive User Interface

## Technologies Used

* Python
* Flask
* Ollama
* HTML
* CSS
* ReportLab

## Installation

Clone the repository:

git clone https://github.com/nandinisingh156/AI-story-forge.git

Install dependencies:

pip install -r requirements.txt

Run Ollama:

ollama serve

Start application:

python app.py

## Usage

1. Enter Character Name
2. Enter Theme
3. Select Genre
4. Select Story Type
5. Click Generate Story
6. Download PDF
